# 料理急先鋒 - 遊戲食材與合成設計 v1.0

## 1. 系統概述

本文件定義了「料理急先鋒」中的食材、加工流程與最終的料理合成配方。設計核心圍繞著不同「工作站」的特定功能，玩家需要依序在各站點對食材進行處理，最終組裝成完整的訂單。

---

## 2. 核心工作站與動作 (簡化流程)

為了簡化遊戲流程、加快節奏，我們將取消「前處理站」。原有的切菜、混合等動作將被整合到烹飪或組裝的過程中，成為隱含操作。

1.  **烹飪站 (Cooking Station)**
    *   `烤 (Grill/Bake)`: 用於烤肉、烘烤麵包或餅皮。
    *   `煎 (Pan-fry)`: 用於煎蛋、肉排或拌炒。
    *   `炸 (Deep-fry)`: 用於炸薯條、炸肉排。
    *   `煮 (Boil)`: 用於煮飯、麵或燙熟食材。

2.  **組裝站 (Assembly Station)**
    *   `組裝 (Assemble)`: 將多個加工或烹飪完成的食材結合成最終料理。這是完成一道菜的最後步驟。

### 4.3 烹飪狀態系統 (Cooking State System)

為了增加烹飪過程的技巧與挑戰性，許多食材的烹飪結果會依據「時間」而產生不同狀態。這需要玩家更精準地掌握時機。

*   **狀態範例 (以牛肉餅為例):**
    1.  **生 (Raw)**: `牛肉餅 [牛]` - 初始狀態。
    2.  **半熟 (Undercooked)**: `半熟牛肉餅 [牛~]` - 烹飪時間不足。可用，但會扣分。
    3.  **完美 (Perfectly Cooked)**: `完美烤牛肉餅 [牛*]` - 在精準的時間內完成烹飪。能獲得額外分數加成。
    4.  **正常 (Cooked)**: `烤牛肉餅 [牛=]` - 在正常時間範圍內完成。標準分數。
    5.  **過熟 (Overcooked)**: `過熟牛肉餅 [牛+]` - 烹飪時間稍長。可用，輕微扣分。
    6.  **烤焦 (Burnt)**: `烤焦物 [x]` - 烹飪時間過長。不可用，必須丟棄，浪費食材。

*   **通用狀態標示建議:**
    *   `~`: 半熟 (Undercooked)
    *   `*`: 完美 (Perfect)
    *   `=`: 正常 (Cooked)
    *   `+`: 過熟 (Overcooked)
    *   `x`: 烤焦 (Burnt)

## 5. 最終組裝與出餐 (Final Assembly & Serving)

此章節是遊戲的核心，定義了玩家如何將烹飪好的食材，透過一個全新的、更具互動性的「出餐盤」機制，組裝成最終的料理。

### 5.1 「出餐盤」機制 (The "Serving Plate" Mechanic)

當一個訂單（例如：`豪華牛肉漢堡`）出現時，遊戲的核心區域會出現一個對應的「出餐盤」。這個盤子直觀地顯示了完成這道菜所需的**最終合成步驟**。

*   **機制流程**:
    1.  **顯示題目**: 盤子會顯示如 `[起司漢堡] + [烤麵包片] → [豪華牛肉漢堡]` 的空格。
    2.  **玩家放置**: 任何玩家都可以從自己的物品欄中，將正確的物品（例如 `起司漢堡 [堡+]`）拖放到對應的空格上。
    3.  **協作完成**: 團隊成員需要協作，將所有必需的物品都放到盤子上。
    4.  **交卷出餐**: 當所有空格都被填滿後，「出餐」按鈕會亮起。點擊後，系統會驗證組合是否正確。
    5.  **結果**:
        *   **正確**: 獲得分數，完成訂單。
        *   **錯誤**: 盤子上的所有物品被丟棄，並扣除分數。

*   **5.1.1 訂單規則 (Order Rules)**
    *   **單一訂單流程 (Single Order Flow)**: 在任何時候，團隊都只能專注於**一道**訂單。只有在當前訂單被成功送出或被放棄後，系統才會生成下一張新訂單。
    *   **放棄訂單選項 (Abandon Order Option)**: 遊戲中提供「放棄訂單」的功能。玩家可以選擇放棄當前的困難訂單，但會立即**扣除一定分數**作為懲罰。放棄後，當前的出餐盤會被清空，並立即生成一張新的訂單。

*   **介面模擬**:
    ```
    +-------------------------------------------------+
    |  訂單: 豪華牛肉漢堡 [堡*]       [放棄訂單 X]  |
    |                                                 |
    |  [堡+]      +      [麵=]      ->      [ 出餐 ]   |
    | (可放置區域) (可放置區域)          (點擊提交)   |
    +-------------------------------------------------+
    ```
